## Categories explained

- **OK** — returns 2xx/3xx, link works.
- **OK_FALSE_POSITIVE** — initial check failed (HEAD method blocked) but GET works. Counted as fine.
- **BROKEN_4xx** — server returns 404/410 etc. The page is gone or never existed.
- **BROKEN_5xx** — server returns 503/500. Mostly bibleodyssey.org pages removed in their 2024 redesign.
- **NETWORK** — DNS / connection error. Site is unreachable or domain is wrong (e.g. typo'd `intertextalbible.com`).
- **WRONG_LIBRARY** — Sefaria URL pointing at a New Testament book. Sefaria is a Jewish-text library and does not host the NT, so these links can never resolve.
- **OFF_TOPIC** — points at theoi.com, a Greek-mythology archive that has no business being cited as a Christian study resource.
- **UNVERIFIED_BOT_BLOCK** — site (British Museum, Met, Penn Museum, Cambridge) returns 403/429 to non-browser requests. Probably works in a real browser; needs manual spot-check.
